# AI Personal Carbon Footprint Tracker 🌍

This project is an AI-powered web application that helps users track, analyze, and reduce their carbon footprint.

## Features

- Carbon emission calculator
- AI prediction of future emissions
- Sustainability score
- Eco challenges and rewards
- Habit tracker
- Dashboard analytics

## Tech Stack

Python  
Streamlit  
SQLite  
Scikit-learn  

## Run the Project

Install dependencies

pip install -r requirements.txt

Run application

Python -m streamlit run app.py
